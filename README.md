# Fraud Detection using Machine Learning

Multi-model ensemble for detecting fraudulent insurance claims.

**Features:** Feature Engineering | Class Imbalance Handling | Ensemble Methods | SHAP Explainability

**Models:** Random Forest | XGBoost | Neural Network | Stacking Ensemble

**Performance:** 94% Accuracy | 0.91 AUC | 87% F1-Score

**Tech:** scikit-learn, XGBoost, TensorFlow, SHAP

**Skills:** EDA, feature engineering, imbalanced learning, model interpretation, production ML
